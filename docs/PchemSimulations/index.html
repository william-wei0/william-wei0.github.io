Classical Angular Momentum of a Rigid Rotor

This is a simulation of the rigid rotor of a two particle system. Classically, angular momentum can have any angle for theta, however, in quantum mechanics, theta is restricted to discrete, specific values of theta. In this simulation you are able to set the angles phi and theta to visualize how the angular momentum is defined classically and compare it to quantized angular momentum

Superposition of two particles in a box

The time-dependent wave function of two particles in a box is often difficult to visualize and graph. Here, you are able to set the energy of two particles and visualize the wave functions. You are also able to set the proportion of each particle and observe how that affects their superposition.

Eigenfunction Approximation of a Triangle Wave

Periodic functions such as sine waves, cosine waves, square waves, and triangle waves can be approximated using an infinite expansion of eigenfunctions. This program allows you to vary the number of terms in the expansion and observe how the number of terms used affects the approximation.
